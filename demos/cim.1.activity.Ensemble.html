Description Six element ensemble, all using three different types of initialisation: r1i1p1 - standard initialisation from previous HadGEM2-A run (not submitted) r1i2p1- atmosphere only initialisation from same run as r1i1p1 r2i3p1 - initialisation from r1i2p1 (2001) - atmosphere and land surface variation r3i2p1 - initialisation from r1i2p1 (2001) - atmosphere variation only r4i3p1 - initialisation from r3i2p1 (2008) - atmosphere and land surface variation r5i3p1 - initialisation from r4i3p1 (2008) - atmosphere and land surface variation
